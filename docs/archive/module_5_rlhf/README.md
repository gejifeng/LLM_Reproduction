### 核心数学公式

**GRPO 目标函数：**
```
L(θ) = -E[(r - b) * log π_θ(a|o)]
```
其中：
- r: 奖励 (reward)
- b: 基线 (baseline，通常是组内平均)
- π_θ: 策略模型
- o: 提示 (prompt)
- a: 回答 (answer)

**KL 散度约束：**
```
L(θ) = -E[(r - b) * log π_θ(a|o)] - β * KL(π_θ || π_ref)
```
防止策略模型偏离参考模型太远

### 奖励函数设计

| 奖励类型 | 条件 | 奖励值 |
|----------|------|--------|
| 格式正确 | 包含 `<answer>` 标签 | +0.1 |
| 答案正确 | 与标准答案匹配 | +1.0 |
| 格式错误 | 缺少必要标签 | -0.1 |
| 答案错误 | 与标准答案不匹配 | 0.0 |
