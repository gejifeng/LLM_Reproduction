### 核心数学公式

**AdamW 更新规则：**
```
m_t = β₁ * m_{t-1} + (1 - β₁) * g_t
v_t = β₂ * v_{t-1} + (1 - β₂) * g_t²
m_hat = m_t / (1 - β₁^t)
v_hat = v_t / (1 - β₂^t)
θ_t = θ_{t-1} - α * m_hat / (√v_hat + ε) - α * λ * θ_{t-1}
```
其中：
- g_t: 梯度
- m_t: 一阶矩估计 (动量)
- v_t: 二阶矩估计 (自适应学习率)
- λ: weight decay

**Cosine Annealing 学习率调度：**
```
lr(t) = lr_min + 0.5 * (lr_max - lr_min) * (1 + cos(π * t / T))
```
其中 T 为总步数，t 为当前步数

**梯度裁剪：**
```
if ||g|| > max_norm:
    g = g * max_norm / ||g||
```
